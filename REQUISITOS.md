# Requisitos



## Exemplo da Estrutura de Requisitos

Lista de Stakeholders (Personas).

- No mínimo 10 Requisitos Funcionais (RF) (O que o sistema faz? Ex: "Paciente deve poder...").

- No mínimo 5 Requisitos Não Funcionais (RNF) (Como o sistema é? Ex: "Sistema deve ser rápido...").

- No mínimo 3 Regras de Negócio (RN) (Políticas da clínica. Ex: "Paciente só
pode...").
